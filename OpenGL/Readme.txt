This program is created by Michael Tran.

Use the WASDQE keys to move the cube in the middle of the screen.
Press C to hide your cursor.
Press V to enable to disable wireframe.
Press ENTER to edit the username which when the application is opened will show as the text "Username" on screen.
When editing user is enabled, the screen will be coloured red.
Press ENTER again to disable editing the username.
Hover your cursor over the username to change its colour.

This program utilises the following additional features from the assessment:
 - Geometry shader to duplicate a text mesh to create the scrolling text effect
 - A point light as represented by the white cube illuminates all surrounding meshes in the scene
 - A specular texture for the meshes that reflects light
